Sobre o Projeto
Este projeto é uma aplicação de agenda de contatos desenvolvida com Django. Ele permite que os usuários criem, visualizem, editem e excluam contatos, além de oferecer uma funcionalidade de pesquisa para facilitar a localização de contatos específicos.
O objetivo deste projeto é fornecer uma solução simples e eficiente para gerenciar contatos pessoais ou profissionais, utilizando as melhores práticas do desenvolvimento web com Django.
O projeto foi desenvolvido por Everton Espedito.
